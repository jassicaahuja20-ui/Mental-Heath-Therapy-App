# 💙 MindEase - Mental Health Therapy App

A simple yet meaningful web application designed to provide emotional support through comforting responses.

---

## 🌿 About the Project
This app allows users to express how they feel and receive gentle, supportive messages in return.  
It creates a safe digital space where emotions are acknowledged and validated.

---

## 🚀 Features
- 📝 Write your thoughts and feelings  
- 💬 Get instant supportive responses  
- 🌸 Clean and calming interface  
- ❤️ Emotion-based interaction  

---

## 🛠️ Built With
- HTML  
- JavaScript  

---

## 💡 Purpose
Built to explore how technology can support mental well-being in a simple and accessible way.
